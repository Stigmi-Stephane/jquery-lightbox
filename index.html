jquery-lightbox by avioli

A jQuery plugin, inspired and based on Lightbox 2 by Lokesh Dhakar

jQuery lightBox plugin is simple, elegant, unobtrusive, no need extra markup and is used to overlay images on the current page through the power and flexibility of jQuery's selector. lightBox is a plugin for jQuery. It was inspired in Lightbox JS by Lokesh Dhakar.

Dependencies

jquery

Install

Copy jquery.lightbox.min.js, jquery.lightbox.css and the images from the images folder to the proper locations of your project. Use ``` $(function() { var settings = {fixedNavigation:true}; $('#gallery a').lightBox(settings); }); ```
